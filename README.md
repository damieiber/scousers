# FanNews - AI-Powered Fan Hub

Una plataforma de noticias deportivas curada por IA para los fanáticos de Liverpool y Everton.

## Features

- ✅ **Feed de noticias IA** con resúmenes automáticos
- ✅ **Multi-equipo** - Liverpool y Everton
- ✅ **Modo Rival** para seguir al rival
- ✅ **Theming dinámico** - colores del equipo del usuario
- ✅ **Bilingüe** - Español e Inglés
- ✅ **Light/Dark mode**

## Tech Stack

- **Next.js 15** (Turbopack)
- **TypeScript**
- **MongoDB** + Mongoose
- **Google Gemini AI**
- **Tailwind CSS** + shadcn/ui
- **next-auth** (Google OAuth + Credentials)

## Getting Started

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
